Seletores CSS
CSS é um conjunto de regras que determinam como o browser deve apresentar o seu conteúdo
universal O seletor universal aplica o estilo a todos os elementos da página. Não utilizei.
tipo O seletor de tipo usa o nome do marcador a estilizar. Não utilizei.
classe O seletor de classe associa um nome a um estilo. Não utilizei.
id O atributo id permite identificar univocamente um elemento do HTML. Não utilizei.
atributo O seletor atributo é a plicadoo a atributos de elementos. Não utilizei.
pseudo-classe Seletor associado ao estado de um elemento. Não utilizei.
pseudo-elemento O seletor pseudo-elemento aplica certas condições a partes de elementos. Não utilizei.
Combinadores de seletores CSS
descendentes O combinador de seletores descendente corresponde a todos os elementos que são descendentes de um elemento especificado. Não utilizei.
filhos O combinador de seletores filho seleciona todos os elementos que são os filhos imediatos de um elemento especificado. Não utilizei.
adjacentes O combinador de seletores irmão adjacente seleciona todos os elementos que são os irmãos adjacentes de um elemento especificado. Não utilizei.
imediatamente adjacentes O seletor de irmão em geral seleciona todos os elementos que são irmãos de um elemento especificado. Não utilizei.
agrupados O combinador de selectores agrupados, agrupa os selectores
